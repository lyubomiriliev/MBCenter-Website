# Fonts for PDF Generation

To fix Cyrillic text rendering in PDFs, download these font files:

1. **NotoSans-Regular.ttf** - Download from: https://fonts.google.com/noto/specimen/Noto+Sans
2. **NotoSans-Bold.ttf** - Download from: https://fonts.google.com/noto/specimen/Noto+Sans

Place both files in this directory (`public/fonts/`).

Alternatively, you can download from:
- https://github.com/google/fonts/raw/main/ofl/notosans/NotoSans-Regular.ttf
- https://github.com/google/fonts/raw/main/ofl/notosans/NotoSans-Bold.ttf
